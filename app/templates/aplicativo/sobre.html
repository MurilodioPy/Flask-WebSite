Controle de Empréstimo de Chaves - Projeto PyKeyLoan

O PyKeyLoan é um projeto desenvolvido para simplificar e organizar o controle de empréstimo de chaves em diversos ambientes. Com uma abordagem eficiente e intuitiva, o sistema permite que usuários registrem empréstimos, devoluções e consultem o status de chaves de forma centralizada.

Tecnologias Utilizadas:

Flask: O PyKeyLoan utiliza o framework web Flask, construído em Python. Flask é conhecido por sua simplicidade e flexibilidade, proporcionando um ambiente de desenvolvimento ágil e eficaz.

Python: A linguagem de programação principal do projeto é o Python. Com uma sintaxe clara e poderosa, Python é ideal para o desenvolvimento web e integração com bancos de dados.

MySQL: Para armazenamento de dados, o PyKeyLoan utiliza o MySQL, um sistema de gerenciamento de banco de dados relacional. O MySQL oferece confiabilidade e desempenho para manipulação eficaz dos dados do projeto.

CSS (Cascading Style Sheets): A camada de apresentação do PyKeyLoan é estilizada utilizando CSS, proporcionando uma interface de usuário atraente e responsiva. O design é pensado para facilitar a usabilidade e tornar a experiência do usuário mais agradável.

Docker: O PyKeyLoan é empacotado e distribuído usando contêineres Docker. Isso simplifica a implantação do sistema em diferentes ambientes, garantindo consistência e facilitando a manutenção.

Funcionalidades Principais:

Registro de Empréstimos: Usuários podem registrar empréstimos de chaves, incluindo informações como data, responsável pelo empréstimo e chave específica.

Consulta de Status: O sistema permite a consulta rápida do status de uma chave, indicando se está disponível, emprestada ou reservada.

Devolução Simples: A devolução de chaves é simplificada, com a opção de registrar qualquer dano ou observação relevante.

Histórico de Transações: Um histórico detalhado de todas as transações, incluindo empréstimos e devoluções, é mantido para referência futura.
